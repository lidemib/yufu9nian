# 渔夫9年餐饮知识库网站 - 部署指南

## 网站文件位置

所有网站文件在 `outputs/` 目录：
- `index.html` - 首页（含引流入口）
- `style.css` - 样式
- `articles/` - 11篇知识库文章（HTML）
- `cover-prompts.md` - 封面图生成命令

## 方法一：GitHub Pages 部署（推荐，免费）

### 1. 安装 Git
下载：https://git-scm.com/download/win

### 2. 创建 GitHub 仓库
- 打开 https://github.com/new
- 仓库名：`yufu9nian` 或 `canyin-zhishi`
- 选 Public（公开）
- 不要勾选任何初始化选项

### 3. 推送网站文件

```powershell
cd C:\Users\Dell\Documents\Codex\2026-06-22\ai-9-2026-06-20-ai\outputs

# 初始化 git
git init
git checkout -b main

# 添加所有网站文件
git add index.html style.css articles/ cover-prompts.md

# 提交
git commit -m "渔夫9年餐饮知识库网站 v1.0"

# 关联远程仓库（替换成你的仓库地址）
git remote add origin https://github.com/你的用户名/yufu9nian.git

# 推送
git push -u origin main
```

### 4. 启用 GitHub Pages
- 打开仓库 → Settings → Pages
- Source: Deploy from a branch
- Branch: main, / (root)
- 点 Save
- 等1-2分钟，拿到网址：`https://你的用户名.github.io/yufu9nian/`

### 5. 绑定自定义域名（可选）
- 买域名：阿里云/GoDaddy，.com 约 100元/年
- 建议域名：yufu9nian.com
- Settings → Pages → Custom domain → 填入域名
- 域名DNS添加CNAME记录指向 `你的用户名.github.io`

## 方法二：Vercel 部署（更简单）

### 1. 注册 Vercel
打开 https://vercel.com ，用 GitHub 账号登录

### 2. 导入仓库
- New Project → 选你的 GitHub 仓库
- Framework: 选 Other
- 直接 Deploy
- 拿到网址：`https://xxx.vercel.app`

## 本地预览

直接在浏览器打开 `outputs/index.html` 查看效果。

## 部署后要做的

1. 把网址发到你的头条/百家号/抖音——引流
2. 网站首页的引流入口目前是文字引导（合规），后期可以加图片
3. 持续更新文章：把新写的头条文章同步到网站
4. 用 cover-prompts.md 里的命令生成封面图，替换占位
